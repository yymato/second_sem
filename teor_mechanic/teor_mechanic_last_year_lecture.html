📌 Тема 1. Кинематика точки и твёрдого тела

Задачи кинематики: описывать движение геометрически, не выясняя причины. Пространство и время в классической механике — абсолютные, независимые.

\[ \vec{v} = \frac{d\vec{r}}{dt} \]
📖 Скорость точки: \(\vec{r}(t)\) — радиус-вектор (положение точки). Производная по времени \(t\) показывает, как быстро меняется положение. Вектор скорости направлен по касательной к траектории.
\[ \vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2} \]
📖 Ускорение точки: характеризует изменение скорости (по модулю и направлению). Вторая производная радиус-вектора.

📈 Движение по окружности: нормальное и тангенциальное ускорение

\[ a_n = \frac{v^2}{R} = \omega^2 R \]
📖 Нормальное (центростремительное) ускорение: всегда направлено к центру окружности. \(v\) — линейная скорость, \(R\) — радиус, \(\omega = \dot{\varphi}\) — угловая скорость. Оно отвечает за изменение направления скорости.
\[ a_\tau = \varepsilon R \]
📖 Тангенциальное ускорение: \(\varepsilon = \dot{\omega}\) — угловое ускорение. Направлено по касательной, меняет модуль скорости.
Рис.1: Вектор скорости \(\vec{v}\) — по касательной, нормальное ускорение \(\vec{a}_n\) — к центру.

🔄 Формулы Эйлера и Ривальса для твёрдого тела

\[ \vec{v} = \vec{\omega} \times \vec{r} \]
📖 Формула Эйлера: для точки тела, вращающегося вокруг оси. \(\vec{\omega}\) — вектор угловой скорости, \(\vec{r}\) — радиус-вектор от оси до точки. Векторное произведение даёт скорость, направленную перпендикулярно \(\vec{\omega}\) и \(\vec{r}\).
\[ \vec{a} = \vec{\varepsilon} \times \vec{r} + \vec{\omega} \times (\vec{\omega} \times \vec{r}) \]
📖 Ривальс (ускорения): первое слагаемое — вращательное ускорение (меняет модуль скорости), второе — осестремительное (центростремительное). \(\vec{\varepsilon} = \dot{\vec{\omega}}\).

📐 Мгновенный центр скоростей (МЦС)

🔹 МЦС — точка плоской фигуры, скорость которой в данный момент равна нулю. Зная МЦС, скорость любой точки: \( v_A = \omega \cdot r_{A/МЦС} \), направлена перпендикулярно радиусу от МЦС.

🌍 Сложение движений. Теорема Кориолиса

\[ \vec{v}_a = \vec{v}_r + \vec{v}_e \]
📖 Сложение скоростей: \(\vec{v}_a\) — абсолютная скорость (относительно неподвижной системы), \(\vec{v}_r\) — относительная (в подвижной системе), \(\vec{v}_e\) — переносная (скорость той точки подвижной системы, где находится точка).
\[ \vec{a}_c = 2\vec{\omega}_e \times \vec{v}_r \]
📖 Кориолисово ускорение: возникает при движении точки во вращающейся системе отсчёта. \(\vec{\omega}_e\) — угловая скорость переносного вращения. Направление по правилу правой руки.

✔ Углы Эйлера: \(\psi\) (прецессия), \(\theta\) (нутация), \(\varphi\) (собственное вращение) — задают ориентацию твёрдого тела с неподвижной точкой.

⚖️ Тема 2. Статика

\[ \vec{M}_O = \vec{r} \times \vec{F} \]
📖 Момент силы относительно точки O: \(\vec{r}\) — радиус из точки O в точку приложения силы. Модуль \( M = F \cdot d\), где \(d\) — плечо (кратчайшее расстояние от O до линии действия силы). Вектор момента перпендикулярен плоскости \(\vec{r}\) и \(\vec{F}\).
\[ M = F \cdot h \]
📖 Момент пары сил: пара — две равные по модулю, противоположно направленные силы. \(h\) — расстояние между линиями действия сил (плечо пары). Момент пары не зависит от выбора точки.
\[ \vec{R} = \sum \vec{F}_i = 0, \quad \vec{M}_O = \sum (\vec{r}_i \times \vec{F}_i) = 0 \]
📖 Условия равновесия произвольной пространственной системы сил: главный вектор (сумма всех сил) и главный момент (сумма моментов) должны равняться нулю. Только тогда тело не движется поступательно и не вращается.

✔ Статический инвариант: \( \vec{R} \cdot \vec{M}_O = const\) — скалярное произведение не зависит от выбора центра приведения. Динамический винт — случай, когда \(\vec{R} \parallel \vec{M}_O\). Трение: сухое \(F_{тр} \le \mu N\), вязкое \(F = -kv\), трение качения \(M_{тр} = \delta N\).

📐 Тема 3. Геометрия масс

\[ \vec{r}_C = \frac{\sum m_i \vec{r}_i}{\sum m_i} \]
📖 Центр масс: средневзвешенное положение всех масс системы. Если тело однородное, центр масс совпадает с геометрическим центром.
\[ I = \sum m_i r_i^2 \]
📖 Момент инерции относительно оси: сумма произведений каждой массы на квадрат расстояния до оси. Чем больше \(I\), тем труднее раскрутить тело.
\[ I = I_C + m d^2 \]
📖 Теорема Штейнера (Гюйгенса-Штейнера): \(I_C\) — момент инерции относительно оси, проходящей через центр масс, \(d\) — расстояние между осями. Позволяет вычислять момент инерции относительно любой оси.
🧠 Тензор инерции: матрица 3×3, компоненты которой учитывают распределение масс относительно всех осей. Диагональные элементы — осевые моменты, недиагональные — центробежные.

⚡ Тема 4. Динамика

\[ \vec{F} = m\vec{a} \]
📖 Второй закон Ньютона: сила \(\vec{F}\) сообщает массе \(m\) ускорение \(\vec{a}\). Основа динамики точки.
\[ \frac{d\vec{Q}}{dt} = \sum \vec{F}^{(e)} \]
📖 Теорема об изменении количества движения: \(\vec{Q}=M\vec{v}_C\) — количество движения системы. Производная по времени равна сумме внешних сил.
\[ M\vec{a}_C = \sum \vec{F}^{(e)} \]
📖 Теорема о движении центра масс: центр масс системы движется как материальная точка, в которой сосредоточена вся масса, под действием всех внешних сил.
\[ \frac{d\vec{L}_O}{dt} = \vec{M}_O^{(e)} \]
📖 Теорема моментов (изменение кинетического момента): \(\vec{L}_O = \sum \vec{r}_i \times m_i\vec{v}_i\) — момент импульса. Скорость его изменения равна главному моменту внешних сил.
\[ T = \frac{1}{2}Mv_C^2 + \frac{1}{2} I_C \omega^2 \]
📖 Теорема Кёнига: кинетическая энергия системы = энергия поступательного движения центра масс + энергия движения относительно центра масс.

✔ Динамические уравнения Эйлера для тела с неподвижной точкой: \(I_1\dot{\omega}_1 + (I_3-I_2)\omega_2\omega_3 = M_1\) (циклическая перестановка). Теория гироскопов: гироскопический момент \(\vec{M} = \vec{\omega} \times \vec{L}\).

📖 Тема 5. Аналитическая динамика

🔹 Виртуальные перемещения \(\delta \vec{r}_i\) — воображаемые, бесконечно малые, согласованные со связями. Принцип виртуальных перемещений: \(\sum \vec{F}_i \cdot \delta \vec{r}_i = 0\) — условие равновесия.
\[ \frac{d}{dt}\left( \frac{\partial T}{\partial \dot{q}_k} \right) - \frac{\partial T}{\partial q_k} = Q_k \]
📖 Уравнения Лагранжа 2-го рода: \(q_k\) — обобщённые координаты, \(T\) — кинетическая энергия, \(Q_k\) — обобщённые силы. Позволяют получить уравнения движения в любых координатах.
\[ L = T - \Pi, \quad \frac{d}{dt}\frac{\partial L}{\partial \dot{q}_k} - \frac{\partial L}{\partial q_k} = 0 \]
📖 Лагранжиан консервативной системы: \(L\) = кин. энергия минус потенциальная. Уравнения Лагранжа выводят динамику из вариационного принципа.
\[ \dot{q}_k = \frac{\partial H}{\partial p_k}, \quad \dot{p}_k = -\frac{\partial H}{\partial q_k} \]
📖 Канонические уравнения Гамильтона: \(H = \sum p_k\dot{q}_k - L\) — функция Гамильтона (энергия), \(p_k = \partial L/\partial \dot{q}_k\) — обобщённые импульсы. Система симметрична и удобна для теоретического анализа.

✔ Устойчивость по Лагранжу-Дирихле: если в равновесии потенциальная энергия имеет минимум, то равновесие устойчиво. Малые колебания: \(\ddot{q} + \omega_0^2 q = 0\) — гармонический осциллятор.
